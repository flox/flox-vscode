# Flox VSCode Extention

## Quick Start
Ensure you have "Extension Test Runner" installed in your VSCode.
It should be a recommended extension for a blank VSCode installation.

```console
$ cd git clone git@github.com/flox/flox-vscode.git;
$ cd flox-vscode;

# Enter Dev Shell
$ flox activate

# Lint 
$ npm run compile

# Run the tests
$ npm run test

# Build
$ npm run compile
```
